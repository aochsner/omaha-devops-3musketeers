# 02-spring-boot

## Prerequisites

- [Docker](https://www.docker.com/)
- [Compose](https://docs.docker.com/compose/)
- [Make](https://www.gnu.org/software/make/)

## Usage

```bash
$ ENVFILE=.env.example make envfile
$ make test
$ make build
$ make run
$ make dockerBuild
$ make dockerRun
$ make clean
```
